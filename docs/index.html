Buttons have several combinable variables to use for complete customization.

<button class="btn btn-md">Medium</button>
<button class="btn btn-lg">Large</button>
<button class="btn btn-xl">Extra Large</button>

Border-Radius

<button class="btn btn-md btn-br-1">Level 1</button>
<button class="btn btn-md btn-br-2">Level 2</button>
<button class="btn btn-md btn-br-3">Level 3</button>
<button class="btn btn-md btn-br-4">Level 4</button>

Solid/Outline

Animations

<button class="btn btn-md btn-ani-raise">Raise</button>
